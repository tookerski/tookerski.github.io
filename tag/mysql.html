    Ubuntu16.04安装MySQL数据库

    1. 首先，通过apt命令本机安装MySQL服务器和客户端 sudo apt-get install mysql-server sudo apt-get install mysql-client
    2. 一旦安装完成，MySQL 服务器应该自动启动。您可以在终端提示符后运行以下命令来检查 MySQL 服务器是否正在运行： sudo netstat -tap | grep mysql 当您运行该命令时，您可以看到类似下面的行： tcp 0 0 localhost.localdomain:mysql *:* LISTEN -

    安装MySQL-connector-java

    1. 解压MySQL-connector-java tar -zxvf mysql*
    2. 复制jar文件至Java安装目录下的lib文件夹 cp mysql* ~/apache/jdk1.7.0_80/lib
