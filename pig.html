Hadoop1.0.4下安装pig0.12.0

周二 26 七月 2016
By tookerski
  1. 使用自带Firefox下载并解压pig-0.12.0.tar.gz在当前目录 tookerski@tookerski:~/Downloads$ tar -zxvf pig*
  2. 复制到home目录下的apache文件夹，注意使用sudo命令 tookerski@tookerski:~/Downloads$ sudo mv pig-0.12.0 ~/apache/
  3. 设置环境变量，用vim打开home目录下.bashrc文件 tookerski@tookerski:~/apache$ sudo vim ~/.bashrc
  4. 环境变量设置如下： export PIG_HOME=/home/tookerski/apache/pig-0.12.0 export PIG_CLASSPATH=$HADOOP_HOME/conf export PATH=$PATH:$PIG_HOME/bin
  5. 使环境变量生效 source ~/.bashrc
  6. 使用pig -x local检查本地运行是否成功 tookerski@tookerski:~/apache$ pig -x local
  7. 出现以下讯息表示安装成功： Warning: $HADOOP_HOME is deprecated. 2016-07-26 23:47:09,722 [main] INFO org.apache.pig.Main - Apache Pig version 0.12.0 (r1529718) compiled Oct 07 2013, 12:20:14 2016-07-26 23:47:09,723 [main] INFO org.apache.pig.Main - Logging error messages to: /home/tookerski/apache/pig_1469548029718.log 2016-07-26 23:47:09,848 [main] INFO org.apache.pig.impl.util.Utils - Default bootup file /home/tookerski/.pigbootup not found 2016-07-26 23:47:10,252 [main] INFO org.apache.pig.backend.hadoop.executionengine.HExecutionEngine - Connecting to hadoop file system at: file:/// grunt>
